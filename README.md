# Shared Contracts

Common DTOs, events, and payload definitions for all PetShop microservices.

## Description
Provides shared models used across multiple services to ensure consistent API contracts and reduce duplication.

## Tech Stack
- Java 17  
- Maven  

## Usage
Imported by individual microservices as a shared dependency.
